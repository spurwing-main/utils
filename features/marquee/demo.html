Marquee Feature Demo Use ?utils-debug=marquee for logs

Usage

Add data-marquee to a container. The children become one unit that is duplicated to fill space and animated horizontally using transform keyframes. The feature honors prefers-reduced-motion with a static variant. See README for full docs.
